Automatically generate compelling excerpts for your posts and pages using AI technology.

== Description ==

WP AI Excerpt uses OpenAI's powerful language models to automatically generate engaging excerpts for your WordPress posts and pages. Save time and improve your content's discoverability with AI-powered summaries.

= Features =

* Generate excerpts using GPT-3.5 Turbo, GPT-4, or GPT-4 Turbo
* Configurable default excerpt length (system-wide setting)
* Per-post excerpt length customization
* Simple one-click excerpt generation
* Support for all post types that support excerpts
* Clean and intuitive interface
* Secure API key storage

= Requirements =

* WordPress 5.0 or higher
* PHP 7.2 or higher
* OpenAI API key

== Installation ==

1. Upload the `wp-ai-excerpt` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > AI Excerpt to configure your OpenAI API key
4. Set your preferred default excerpt length
5. Start generating excerpts in the post editor!

== Frequently Asked Questions ==

= How do I get an OpenAI API key? =

You can sign up for an OpenAI account and get your API key at https://platform.openai.com/

= Which AI models are supported? =

The plugin supports GPT-3.5 Turbo, GPT-4, and GPT-4 Turbo models.

= Can I customize the excerpt length for individual posts? =

Yes! While there's a system-wide default length setting, you can specify a different length when generating an excerpt for any individual post.

= Does this work with custom post types? =

Yes, the plugin works with any post type that supports excerpts.
